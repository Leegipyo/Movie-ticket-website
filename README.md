## 📧 이메일 인증번호 외부 API(라이브러리)을 사용
- 회원가입시 사용자가 작성한 이메일로 인증번호를 발송하기 위해서, 서버에서 인증번호 랜덤으로 생성 후 javax.mail API를 사용하여 Gmail SMTP로 사용자의 이메일로 인증번호 전송

## 💥 이슈사항 및 해결
- 공공기관 API에서 제공하는 첨부된 예고 영상의 URL로 Media를 페이지에서 활용하고자 했으나, 해당 미디어 서버는 Cross-Origin을 허용하지 않아 유튜브 영상으로 대체
